# CPU's Register

Each Register has a 16-Bit with Lower 8-Bit and Higher 8-Bit

- **AF**
  - Low: A (Accumulator)
  - High: F (Flag)
- **BC**
  - Low: B
  - High: C
- **DE**
  - Low: D
  - High: E
- **HL**
  - Low: H
  - High: L
- **SP: Stack Pointer**
  - Low: -
  - High: -
- **PC: Program Counter / Pointer**
  - Low: -
  - High: -
